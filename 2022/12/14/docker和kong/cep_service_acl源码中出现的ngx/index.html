NGX

ngx.re.gmatch

先写一下ngx.re.match

作用:在字符串中根据正则表达式find target

gmatch类似match，但是match返回一个

ngx.ctx

ngx.ctx.route.id

是一个表，提供插入查询

ngx()

ngx.req.get_headers()[]

syntax: headers = ngx.req.get_headers(max_headers?, raw?)

不带[]时返回一个table，即所有满足的结果表

带[String]时，返回满足String的header

ngx.req.set_header()

example

ngx.req.set_header(“Content-Type”, “text/css”)

ngx.req.set_header(“Foo”, {“a”, “abc”})

当所选header的值为nil时，等价于clear_header

ngx.req.set_header(“X-Foo”, nil) == ngx.req.clear_header(“X-Foo”)

ngx.req.get_uri_args()[]

与get_header用法相似

返回一个table,所有满足条件的 URL query中的参数

ngx.req.clear_header(header_name)

清除满足header_name的header

ngx.time()

Returns the elapsed seconds from the epoch for the current time stamp from the nginx cached time
